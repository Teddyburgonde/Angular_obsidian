Components: 
C'est un bloc d'interface réutilisable avec sa logique, son html et son style.
IL est composer de 4 fichiers:
html -> html
scss -> css
ts -> La logique du composant
spec -> Pour les tests unitaires

Service:
IL y a toujours @Injectable() pour que la ou les fonctions puissent etre appeler dans un component.
Un service peut appeler un autre service grace a inject(nom du service).
Un service expose une fonction et le component appel la fonction.

Directives: 
Ils permetttent de modifier le comportement ou l'apparence d'un élément html.
Les directives s'utilisent uniquement dans les templates HTML.
Les 3 principales:
	@if -> Affiche un élément si une condition est vraie
		@if(condition){
			...
		}
	@for -> répete un élement pour chaque item d'un tableau
		@for (item of items; track item){
			...
		}
	
	@switch -> Affiche un élément selon une valeur
		@switch (variable) {
  			@case ('valeur1') {
    			  <p>cas 1</p>
  			}
  			@case ('valeur2') {
    		 	 <p>cas 2</p>
  			}
  			@default {
    		  	 <p>cas par défaut</p>
  			}
		}
		
Pipes: 
Transforme l'affichage d'une donnée dans le HTML sans modifier la donnée elle-même.
{{ mavariable | nomDuPipe }} 
Les pipes les plus courants: 
uppercase → met en majuscules
lowercase → met en minuscules
date → formate une date
currency → formate un nombre en monnaie
json → affiche un objet en JSON
