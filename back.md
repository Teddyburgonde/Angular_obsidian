## Un back-end web est composé de :

- **Serveur** - traite les requêtes (Tomcat via Spring boot).
- **Base de données** - stocke les données (MySQL, PostgreSQL, MongoDB...).
- **API** - interface de communication entre front et back.
- **Logique métier** - règles, calculs, validations.
- **Authentification** - gestion des utilisateurs et sécurité.
